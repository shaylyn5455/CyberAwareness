1. Project Vision
Aries-X Sentinel is a high-performance terminal assistant built to provide critical cybersecurity education within the South African digital landscape. Designed with a "Solar Sentinel" aesthetic, the system offers an immersive, guided experience for users to learn about data protection and threat prevention.

2. Core Features
Instant Operator Access: Streamlined interface that bypasses traditional login for immediate emergency security briefing.

Guided Conversational Logic: A state-aware chatbot that uses a "Memory System" to lead users through security protocols via Yes/No confirmations.

South African Context: Specialized modules covering the POPI Act (POPIA) and local banking threats like Vishing.

Immersive Terminal UI: A custom WPF theme featuring high-contrast "Solar Orange" accents and terminal-style typography.

Automated Sound Integration: Audio feedback loops triggered via the AppConfig engine for an authentic system experience.

3. Security Briefing Directory
Users can interact with the system using natural language or specific sector codes:

POPIA: Understanding privacy and data rights in SA.

Banking Scams: Awareness regarding OTP fraud and voice phishing.

Mobile Safety: Protocols for physical device security and remote locking.

Phishing Defense: Identifying fraudulent digital communications.

4. Technical Architecture
Language: C#

Framework: .NET 8.0 (Windows Presentation Foundation)

CI/CD: GitHub Actions automated build pipeline (passing).

Logic Engine: Chatbot.cs utilizing a last-topic memory state machine.
